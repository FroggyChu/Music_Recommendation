# 나만의 노래 추천기 (유사도 기반)

이 프로젝트는 Kaggle의 `Spotify Tracks` 타입 데이터셋(원본 저자 © Original Authors, 라이선스: Open Database License (ODbL))을 사용해 만든 간단한 음악 추천 웹페이지입니다. 이 저장소의 dataset.csv는 Kaggle Spotify Tracks 데이터셋에서 제공된 파일 그대로 포함되어 있습니다.
**중요**: 데이터셋은 ODbL(오픈 데이터베이스 라이선스) 하에 배포됩니다. 데이터 또는 가공된 데이터 파일을 재배포할 경우 동일한 ODbL 조건이 적용됩니다.

## 포함된 파일
- `index.html` — 추천 웹 UI 및 로컬에서 dataset.csv를 읽어 동작하는 정적 페이지
- `dataset.csv` — (선택) Kaggle 데이터셋 원본 (ODbL 적용)
- `DATA_LICENSE.md` — 데이터 라이선스 및 출처 정보
- `LICENSE` — 이 저장소의 코드(HTML/CSS/JS)에 적용되는 라이선스 (MIT)
